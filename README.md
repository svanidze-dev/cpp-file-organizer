# C++ File Organizer

Fast and lightweight tool written in C++ for organizing files and folders.

## Features
- List all files in a directory
- Sort files by extension
- Remove duplicate files (coming soon)
- Generate report (coming soon)

## Build

### Windows (MinGW / MSVC)
g++ main.cpp -o organizer

### Linux / Mac
g++ main.cpp -std=c++17 -o organizer

## Usage
./organizer

Enter folder path when asked.
